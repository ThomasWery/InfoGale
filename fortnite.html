ACCUEIL

Sortie Du jeu:

Fortnite Battle Royale est sorti pour la première fois le 25 juillet 2017

Type de jeu:

Fortnite BR est un jeu Battle Royale

Créateur:

Epic Games a crée Fortnite

Description

Ce jeu dont le but est de survivre a 100 joueurs et d'être le dernier sur une l'île connait

un énorme succes: en partie grâce a ces graphismes de style cartoon.

Les Modes de jeu:

Epic Games font énormement de mise a jour par rapport au mode de jeu temporaire en voici quelque un : Gros explosifs, 50vs50, foire d'empoigne, etc. Sinon il y a les modes principaux : le solo, le duo, le section, le créatif, le terrain de jeu

Les armes:

Il y a 5 types d'armes les pistolets mitrailleurs, les snipers, les fusils à pompes, les pistolets, les explosifs et les fusils d'assaults. Chaque arme a une ou plusieurs couleur qui corespond a une rareté : Blanche=Commune, Verte=Inhabituelle, Bleue=Rare, Violette=Epique, Orange=Légendaire
